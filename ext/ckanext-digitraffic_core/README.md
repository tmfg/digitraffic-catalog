# ckanext-digitraffic_core

CKAN extension providing core functionality for Digitraffic.

## Installation

To install ckanext-digitraffic_core:

1. Activate your CKAN virtual environment
2. Install the extension:
   ```
   pip install .
   ```
3. Add `digitraffic_core` to the `ckan.plugins` setting in your CKAN config file
4. Restart CKAN
